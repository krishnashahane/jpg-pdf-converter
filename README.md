# 🗂️ DocX

A web app for converting between JPG, PDF, Word, and PowerPoint formats. Built by **Krishna**.

## Features

- JPG to PDF (batch up to 20 images)
- PDF to JPG (first page)
- Word (DOCX) to PDF
- PowerPoint (PPTX) to PDF
- PDF to Word (DOCX)
- Drag-and-drop upload, 100MB max per file
- Files deleted automatically after conversion
- No sign-up, no watermarks, no ads

## Setup

```bash
npm install
npm start
```

Open `http://localhost:3000`

## Development

```bash
npm run dev
```

## Tech Stack

- Node.js + Express
- pdf-lib, Sharp, Mammoth, Officegen
- Vanilla HTML/CSS/JS frontend
